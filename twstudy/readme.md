# Multiple Pages Website using Html Css Javascript & Bootstrap 5 | Responsive

## Template Features:-
- Based on Bootstrap 5.x
- 6 Pages
- Modern and clean design.
- 100% Responsive
- Live Style Switcher
- 5 Preset Colors (all colors can be customized)
- Light/Dark mode
- Glassmorphism design
